# PlusTap HTML Structure Reference

> 실제 구조는 Phase 2에서 사용자가 제공한 샘플 HTML을 분석하여 추출 예정.
> 추출 시 다음 클래스명/구조를 정확히 따를 것:
>
> - `section.plustab-article`
> - `div.title-box > strong.title`
> - `div.article-content > div.desc-box > p.desc`
> - `div.callout-box`, `div.quote-box`, `div.button-box`
> - B타입은 `div.swiper` 포함
